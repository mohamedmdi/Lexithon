# Lexithon (Mon Vocabulaire)
## _The Duolingo Clone App_


Lexithon or "Mon Vocabulaire" is an Intern project to create a look-like Duolingo QUIZ App in Android


## Instllation
Expo Go App needed to run the build

```sh
npm i
```
```sh
npm start
```
